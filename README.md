# Install
## Termux:
$ pkg install git -y 

$ git clone https://github.com/Texsas-tool/texsas-bomber

$ cd texsas-bomber

$ bash install.sh

$ python texsas-bomber.py

$ git clone https://github.com/biyivi/texsas-bomber
